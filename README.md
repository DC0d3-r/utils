# Utils

Portable utilities and bootstrap skills for setting up new machines.

## Skills

| Skill | What it does |
|-------|-------------|
| [`cli-bootstrap/`](skills/cli-bootstrap/) | Full CLI environment setup — terminal, prompt, tools, theme. Like a senior engineer walking you through their setup. |

## Philosophy

These aren't dotfiles managers. They're **educational scripts** that explain what they install, why, and what alternatives exist. Run them interactively for the guided experience, or `--all --yes` for a quick setup.
