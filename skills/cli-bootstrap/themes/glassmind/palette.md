# Glassmind — Color Reference Card

> Translucent minimal — gray-violet tones, soft contrast, context-aware.

## Background Surfaces

| Name     | Hex       | Best for                              |
|----------|-----------|---------------------------------------|
| Void     | `#101014` | Tab bar background, deepest layer     |
| Ink      | `#15151A` | Inactive tab background               |
| Sumi     | `#1A1A22` | Primary background, ANSI black        |
| Stone    | `#222230` | Selection bg, one elevation up        |
| Pebble   | `#2A2A3A` | Selected range, two elevations up     |
| Drift    | `#353548` | Borders, autosuggestions              |
| Mist     | `#404058` | ANSI bright black, dim UI elements    |
| Fog      | `#4D4D68` | Subtle separators                     |

## Foreground Tones

| Name      | Hex       | Best for                             |

| Ash       | `#7A7A8C` | Comments, inactive text              |
| Clay      | `#9090A0` | Options/flags, graph labels          |
| Parchment | `#A8A8B8` | Secondary text                       |
| Ivory     | `#D8D4EA` | Primary foreground, ANSI white       |
| Silk      | `#EAE8F0` | Selection fg, bright white, emphasis |
