# Wavefront — Color Reference Card

> Japanese woodblock muted elegance. Ink, paper, wave, stone.

## Background Surfaces

| Name     | Hex       | Best for                              |
|----------|-----------|---------------------------------------|
| Void     | `#111117` | Tab bar background, deepest layer     |
| Ink      | `#16161D` | Inactive tab background               |
| Sumi     | `#1F1F28` | Primary background, ANSI black        |
| Stone    | `#2A2A37` | Selection bg, one elevation up        |
| Pebble   | `#363646` | Selected range, two elevations up     |
| Drift    | `#44445B` | Borders, autosuggestions              |
| Mist     | `#54546D` | ANSI bright black, dim UI elements    |
| Fog      | `#65657A` | Subtle separators                     |
