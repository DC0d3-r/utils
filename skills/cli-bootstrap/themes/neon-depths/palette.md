# Neon Depths — Color Reference Card

> Cyberpunk cool purple, high-contrast neon accents against deep indigo.

## Background Surfaces

| Name     | Hex       | Best for                              |
|----------|-----------|---------------------------------------|
| Void     | `#0D0D14` | Tab bar background, deepest layer     |
| Ink      | `#12121C` | Inactive tab background               |
| Sumi     | `#181824` | Primary background, ANSI black        |
| Stone    | `#22222E` | Selection bg, one elevation up        |
| Pebble   | `#2C2C3A` | Selected range, two elevations up     |
| Drift    | `#3A3A4A` | Borders, autosuggestions              |
| Mist     | `#4A4A5C` | ANSI bright black, dim UI elements    |
| Fog      | `#5A5A6E` | Subtle separators                     |

## Foreground Tones

| Name      | Hex       | Best for                             |
|-----------|-----------|--------------------------------------|
| Ash       | `#8080A0` | Comments, inactive text              |
| Clay      | `#9898B8` | Options/flags, graph labels          |
| Parchment | `#B8B8D0` | Secondary text                       |
| Ivory     | `#D0CBE4` | Primary foreground, ANSI white       |
| Silk      | `#E8E4F0` | Selection fg, bright white, emphasis |
